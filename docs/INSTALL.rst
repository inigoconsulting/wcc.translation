wcc.translation Installation
----------------------------

To install wcc.translation using zc.buildout and the plone.recipe.zope2instance
recipe to manage your project, you can do this:

* Add ``wcc.translation`` to the list of eggs to install, e.g.:

    [buildout]
    ...
    eggs =
        ...
        wcc.translation

* Re-run buildout, e.g. with:

    $ ./bin/buildout
